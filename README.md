#  Movie Rental Analytics

## Project Overview

**Movie Rental Analytics** is a Business Intelligence project developed using **MySQL** and **Power BI** to analyze the Sakila DVD Rental database. The project focuses on customer behavior, rental trends, revenue performance, inventory management, film analysis, actor analysis, and geographic insights.

By combining SQL-based Exploratory Data Analysis (EDA) with interactive Power BI dashboards, the project transforms raw transactional data into meaningful business insights and actionable recommendations.

---

#  Objectives

The primary objectives of this project are:

* Analyze customer behavior and rental patterns.
* Identify revenue trends and top-performing locations.
* Evaluate film inventory and category performance.
* Explore actor participation and contribution.
* Generate business insights using SQL and Power BI.
* Support data-driven decision-making through interactive dashboards.

---

#  Tools & Technologies

* **Database:** MySQL (Sakila Sample Database)
* **Visualization:** Microsoft Power BI
* **Query Language:** SQL
* **Documentation:** PDF, Excel, PowerPoint
* **Version Control:** Git & GitHub
